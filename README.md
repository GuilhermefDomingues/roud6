# Cartão Animado Round 6

Projeto desenvolvido com HTML e CSS inspirado no cartão da série Round 6. O card possui efeito de rotação, exibindo uma frente com formas geométricas e um verso com ícone e numeração.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Font Awesome

## ✨ Funcionalidades

- Cartão animado com efeito flip
- Frente com formas geométricas
- Verso com ícone e número
- Animação ao passar o mouse
- Design inspirado em Round 6
- Sem uso de JavaScript

## 📂 Estrutura do Projeto

```bash
Cartao-Animado-Round6/
│
├── index.html
│
├── css/
│   └── style.css
│
└── img/
    └── icone.png
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.

## 📚 Conceitos Praticados

- Estruturação com HTML
- Estilização com CSS
- Efeito flip card
- Transformações CSS
- Transições
- Position Absolute e Relative
- Formas geométricas com CSS
- Organização visual de componentes

## 💡 Objetivo

Praticar animações e transformações em CSS criando um cartão interativo inspirado em uma referência visual popular.
